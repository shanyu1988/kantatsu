# 监控系统说明
### 版本号：0.4

## 项目安装
```
npm install
```

### 启动项目开发命令
```
npm run serve
```

### 编译项目生产命令
```
npm run build
```

### 测试命令
```
npm run test
```

### 语法查错命令
```
npm run lint
```

### 配置文档
[配置项参考文档](https://cli.vuejs.org/config/).
